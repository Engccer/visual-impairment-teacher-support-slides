# 효과음 다운로드 상세 가이드

프레젠테이션에 최적화된 효과음을 선택하는 가이드입니다.

## 1. start.mp3 (프레젠테이션 시작)

**특징:**
- 환영하고 긍정적인 느낌
- 너무 크지 않고 부드러운 톤
- 전문적이고 격식 있는 분위기
- 길이: 1-2초

**추천 검색어:**
- "presentation start"
- "soft bell chime"
- "notification gentle"
- "welcome sound"
- "ding positive"

**구체적 추천 (Freesound.org):**
1. "Ding Sound Effect" - 깔끔한 벨 소리
2. "Soft Notification" - 부드러운 알림음
3. "Welcome Chime" - 환영 차임

**Pixabay 추천:**
- "positive-notification"
- "bell-chime"
- "soft-ding"

---

## 2. next.mp3 (다음 슬라이드)

**특징:**
- 가볍고 경쾌한 클릭/스와이프 소리
- 방해되지 않을 정도로 짧음 (0.5-1초)
- 진행감을 주는 긍정적 효과음
- 반복해서 들어도 거슬리지 않는 소리

**추천 검색어:**
- "button click soft"
- "page turn"
- "swipe sound"
- "light click"
- "ui click gentle"

**구체적 추천 (Freesound.org):**
1. "Soft Button Click" - 부드러운 버튼 클릭
2. "Page Flip" - 페이지 넘기는 소리
3. "UI Swipe" - UI 스와이프 효과음

**Mixkit 추천:**
- "button-click-simple"
- "interface-soft-click"

---

## 3. prev.mp3 (이전 슬라이드)

**특징:**
- next.mp3와 유사하지만 약간 다른 톤
- '뒤로 가기' 느낌을 주는 소리
- 같은 볼륨과 길이 (0.5-1초)
- next.mp3보다 약간 낮은 음높이 추천

**추천 검색어:**
- "back button"
- "reverse click"
- "undo sound"
- "previous button"
- "backward sound"

**구체적 추천 (Freesound.org):**
1. "Back Button Click" - 뒤로가기 버튼 소리
2. "Reverse Beep" - 역방향 신호음
3. "Undo Sound" - 취소 효과음

**Tip:** next.mp3와 같은 효과음을 사용하되, 오디오 편집 프로그램에서 피치를 약간 낮추는 것도 좋은 방법입니다.

---

## 4. end.mp3 (프레젠테이션 종료)

**특징:**
- 완료, 성취감을 주는 소리
- 긍정적이고 따뜻한 마무리
- 약간 긴 여운 (1-3초)
- 박수나 성공 사운드

**추천 검색어:**
- "success sound"
- "achievement complete"
- "applause short"
- "finish positive"
- "completion sound"
- "tada celebration"

**구체적 추천 (Freesound.org):**
1. "Success Fanfare" - 성공 팡파레
2. "Achievement Unlock" - 성취 사운드
3. "Applause Short" - 짧은 박수 소리
4. "Completion Bell" - 완료 벨 소리

**Pixabay 추천:**
- "success-fanfare"
- "achievement-bell"
- "completion-sound"

## 다운로드 사이트 바로가기

### 1. Freesound (추천 ⭐)
- URL: https://freesound.org/
- 특징: 가장 다양한 효과음, 커뮤니티 기반
- 회원가입: 필요 (무료)
- 라이선스: Creative Commons (출처 표시 필요/불필요 다양)

**사용 방법:**
1. 회원가입 및 로그인
2. 검색창에 위 검색어 입력
3. 필터에서 "Short (< 5sec)" 선택
4. 라이선스 확인 (CC0 추천 - 출처 표시 불필요)
5. 다운로드 후 파일명을 start.mp3, next.mp3 등으로 변경

### 2. Pixabay
- URL: https://pixabay.com/sound-effects/
- 특징: 깔끔한 UI, 저작권 걱정 없음
- 회원가입: 불필요
- 라이선스: Pixabay License (자유롭게 사용 가능)

**사용 방법:**
1. Sound Effects 섹션 이동
2. 검색어 입력
3. 미리듣기 후 다운로드
4. 파일명 변경하여 audio 폴더에 저장

### 3. Mixkit
- URL: https://mixkit.co/free-sound-effects/
- 특징: 전문적 품질, UI/UX 효과음 풍부
- 회원가입: 불필요
- 라이선스: Mixkit License (자유롭게 사용 가능)

### 4. Zapsplat
- URL: https://www.zapsplat.com/
- 특징: 방대한 라이브러리
- 회원가입: 필요 (무료)
- 라이선스: 상업적 사용 가능

## 파일 저장 체크리스트

다운로드 후 다음 작업을 완료하세요:

- [ ] 파일을 MP3 형식으로 저장 (다른 형식인 경우 변환 필요)
- [ ] 파일명을 정확히 변경:
  - [ ] start.mp3
  - [ ] next.mp3
  - [ ] prev.mp3
  - [ ] end.mp3
- [ ] 모든 파일을 `audio/` 폴더에 저장
- [ ] 브라우저에서 index.html 열어 효과음 테스트

## 문제 해결

### 효과음이 재생되지 않는 경우:

1. 파일명이 정확한지 확인 (대소문자 구분, 띄어쓰기 없음)
2. 파일이 `audio/` 폴더에 있는지 확인
3. 파일 형식이 MP3인지 확인
4. 브라우저 콘솔(F12)에서 에러 메시지 확인
5. 다른 브라우저에서 테스트

### 효과음을 끄고 싶은 경우:

브라우저 콘솔(F12)에서 입력:
```javascript
presentation.toggleSound()
```

## 추천 조합 예시

### 조합 1: 미니멀한 느낌
- start: 부드러운 딩 (1초)
- next: 짧은 클릭 (0.3초)
- prev: 같은 클릭, 약간 낮은 음
- end: 짧은 성공 벨 (1.5초)

### 조합 2: 격식 있는 느낌
- start: 차임벨 (1.5초)
- next: 페이지 넘기는 소리 (0.5초)
- prev: 같은 소리, 역재생 느낌
- end: 박수 소리 (2초)

### 조합 3: 모던한 느낌
- start: UI 알림음 (1초)
- next: 스와이프 효과음 (0.4초)
- prev: 역스와이프 (0.4초)
- end: 성취 팡파레 (2초)
