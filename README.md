# 시각장애인교원 사례 발표 슬라이드

## 프로젝트 정보

- **제목**: 시각장애인교원 사례 - 신명중학교 사례를 중심으로
- **발표자**: 김헌용 (신명중학교 교사 / 함께하는장애인교원노동조합 위원장)
- **날짜**: 2025년 10월 22일
- **형식**:
  - HTML 기반 인터랙티브 프레젠테이션 (주 버전)
  - PowerPoint (.pptx) 백업 버전
- **슬라이드 수**: 14장 (이미지 슬라이드 2장 포함)

## 슬라이드 구성

1. **표지** - 제목, 발표자 정보, 날짜
2. **15년의 질문** - 도입부, 학생의 질문과 교직 경력 소개
3. **편의시설 지원** - 신명중학교 포괄사업비 1,600만 원 활용 사례
4. **신명중학교 편의시설 개선 (이미지)** ⭐ - 점자안내도, 복도/계단 점자블록 사진
5. **대체 교과서** - 점자 교과서의 중요성 및 신청 방법
6. **업무보조인력 운영 (1)** - 서울시교육청 사업, 예산 및 업무 내용
7. **업무보조인력 운영 (2)** - 성공적 운영의 핵심 원칙
8. **보조공학기술 & 학생 주도 개선** - 고용공단 지원 + 구룡중 사례
9. **학생 주도 환경 개선 (이미지)** ⭐ - 구룡중 민원 및 점자블록 설치 사진
10. **노조 활동 지원** - 타임오프제 활용 사례
11. **좋은 관리자 vs 아쉬운 관리자** - 대조 사례를 통한 교훈
12. **관리자에게 바라는 세 가지** - 핵심 메시지
13. **포용적 학교 문화** - 궁극적 목표
14. **감사합니다** - 마무리 슬라이드

⭐ = 이미지 슬라이드

## 사용 방법

### 🌐 HTML 버전 (권장)

#### 기본 조작

1. `index.html` 파일을 웹 브라우저에서 열기
2. 다음 슬라이드로 이동:
   - 키보드: `→` (오른쪽 화살표), `↓` (아래 화살표), `Space`, `Page Down`
   - 마우스: 화면 하단 `›` 버튼 클릭
   - 터치: 왼쪽으로 스와이프
3. 이전 슬라이드로 이동:
   - 키보드: `←` (왼쪽 화살표), `↑` (위 화살표), `Page Up`
   - 마우스: 화면 하단 `‹` 버튼 클릭
   - 터치: 오른쪽으로 스와이프
4. 첫 슬라이드로 이동: `Home` 키
5. 마지막 슬라이드로 이동: `End` 키

### 추가 기능

- **진행 상황 표시**: 화면 상단의 진행 바로 현재 위치 확인
- **슬라이드 카운터**: 화면 하단에 현재 슬라이드 번호 표시
- **효과음 시스템**: 프레젠테이션 시작, 슬라이드 이동, 종료 시 효과음 재생
  - start.mp3/wav: 프레젠테이션 시작
  - next.mp3/wav: 다음 슬라이드
  - prev.mp3/wav: 이전 슬라이드
  - end.mp3/wav: 마지막 슬라이드
  - 효과음 on/off: 콘솔에서 `presentation.toggleSound()` 실행
- **제목 음성 (TTS)**: 각 슬라이드 제목 자동 읽기 ✅
  - 14개 슬라이드별 제목 음성 (slide01_title.mp3 ~ slide14_title.mp3)
  - 효과음 재생 후 자동으로 제목 읽기
  - 접근성 향상 및 청각적 피드백
- **이미지 슬라이드**: 편의시설 및 학생 주도 개선 사례 사진 포함
- **인쇄 지원**: 브라우저의 인쇄 기능으로 모든 슬라이드 출력 가능
- **반응형 디자인**: 다양한 화면 크기에 자동 적응

### 📊 PowerPoint 버전 (백업)

#### 기본 정보

- **파일명**: `시각장애인교원_사례_발표.pptx`
- **용도**: 백업, 오프라인 발표, 다른 PC에서 사용
- **포함 내용**: 14장 슬라이드, 6장 이미지
- **효과음**: 별도 추가 필요 (가이드 참조)

#### 사용 방법

1. Microsoft PowerPoint 2010 이상에서 열기
2. F5 키를 눌러 슬라이드 쇼 시작
3. 클릭 또는 화살표 키로 슬라이드 전환
4. ESC 키로 슬라이드 쇼 종료

#### 효과음 추가

`PowerPoint_효과음_추가_가이드.md` 파일을 참고하세요.

**간단 요약:**
1. `audio/sound-generator.html`에서 효과음 생성
2. PowerPoint에서 삽입 → 오디오로 효과음 추가
3. 자동 재생 설정

**장점:**

- ✅ 수정 및 편집 쉬움

**단점:**
